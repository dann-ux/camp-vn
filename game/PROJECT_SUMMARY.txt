# PROJECT SUMMARY & DEVELOPMENT LOG

## Current Goal
Create a professional-grade visual novel set in a summer camp where the player (the protagonist) interacts with kids aged 5–11. The game should:
- Feature age-appropriate dialogue and behavior.
- Allow branching relationships (friends, rivals, best friends, enemies).
- Include deep emotional moments tied to relationship states.
- Provide multiple narrative paths based on player choices.
- Include a title screen, character creation, and professional visual novel UI.

## What Has Been Implemented So Far
1. **Character System**
   - `Kid` class with mood, trust, personality, species, and memory tracking.
   - Global `all_kids` list containing the player and other kids.
   - Species selection (human, cat, dog, rabbit, fox, bear).
   - Personality assignment (`shy`, `energetic`, `calm`, `playful`).

2. **Dialogue System**
   - `DIALOGUE_BANK` with context-specific lines for each personality.
   - `choose_dialogue` and `say_kid` functions for dynamic, non-repeating lines.
   - `say_duo` for simultaneous kid dialogue.

3. **Relationship & Memory Systems**
   - `relationships` dictionary tracking positive/negative values.
   - `change_relationship`, `get_relationship`, and `relationship_event` helpers.
   - `memory_log` storing events per kid; summary generation based on positive/negative counts.

4. **Scene & Event Management**
   - `scene_kids`, `choose_event`, and `format_scene` for dynamic kid grouping.
   - Story arc detection (`best_friend_scene`, `rival_scene`) with trigger tracking.
   - Location labels and map screen (`camp_map`) for navigation.

5. **UI & Navigation**
   - Title screen with New Game, Load Game, Settings, and Quit options.
   - Character creation screen with name input and species selection.
   - Settings screen with basic options (placeholder).
   - Camp map navigation screen.

6. **Core Game Loop**
   - `camp_loop` label initializing the camp environment and presenting choices.
   - Menu options for wandering, thinking, and interacting with the environment.
